### <font color="blue">一、霍夫曼编码</font>


给定8个灰度级及其概率分布：

| 灰度级 $r_k$ | 概率 $P_r(r_k)$ |
| :-------: | :-----------: |
|     0     |      0.19     |
|    1/7    |      0.25     |
|    2/7    |      0.21     |
|    3/7    |      0.16     |
|    4/7    |      0.08     |
|    5/7    |      0.06     |
|    6/7    |      0.03     |
|     1     |      0.02     |

#### 1. 霍夫曼编码过程

将灰度级按概率从大到小排列,每次选取概率最小的两个节点合并，形成新的节点，直到只剩一个根节点。

| 步骤 | 合并的两个最小概率节点 | 合并后概率 |
|:---:|:---:|:---:|
| 1 | 0.02 + 0.03 | 0.05 |
| 2 | 0.05 + 0.06 | 0.11 |
| 3 | 0.08 + 0.11 | 0.19 |
| 4 | 0.16 + 0.19 | 0.35 |
| 5 | 0.21 + 0.19 | 0.40 |
| 6 | 0.35 + 0.25 | 0.60 |
| 7 | 0.40 + 0.60 | 1.00 |


#### 2. 霍夫曼编码结果

| 灰度级 $r_k$ | 概率 $P_r(r_k)$ | 霍夫曼编码 | 码长 |
|:---:|:---:|:---:|:---:|
| 0 | 0.19 | 111 | 3 |
| 1/7 | 0.25 | 10 | 2 |
| 2/7 | 0.21 | 01 | 2 |
| 3/7 | 0.16 | 110 | 3 |
| 4/7 | 0.08 | 000 | 3 |
| 5/7 | 0.06 | 0011 | 4 |
| 6/7 | 0.03 | 00101 | 5 |
| 1 | 0.02 | 00100 | 5 |

#### 3. 压缩比和相对冗余

**平均码长**：

$$L_{avg} = \sum_{k=0}^{7} l_k \cdot P_r(r_k) = 3 \times 0.19 + 2 \times 0.25 + 2 \times 0.21 + 3 \times 0.16 + 3 \times 0.08 + 4 \times 0.06 + 5 \times 0.03 + 5 \times 0.02 = 2.70 \text{ bit/symbol}$$
v
**信息熵**：

$$H = -\sum_{k=0}^{7} P_r(r_k) \log_2 P_r(r_k) = 2.6508 \text{ bit/symbol}$$

**原始编码**：8个灰度级需 $\lceil\log_2 8\rceil = 3$ bit。

**压缩比**：

$$CR = \frac{n}{L_{avg}} = \frac{3}{2.70} = 1.1111$$

**编码效率**：

$$\eta = \frac{H}{L_{avg}} = \frac{2.6508}{2.70} = 98.18\%$$

**相对冗余**：

$$R = 1 - \eta = 1 - \frac{H}{L_{avg}} = 1 - 0.9818 = 1.82\%$$

从直方图可以看出，该图像的灰度值主要集中在中间区域，在低灰度（暗部）区域也有一定的分布。

#### 2. 计算理论无损压缩极限（信息熵）

根据灰度概率分布，利用 Shannon 信息熵公式：

$$H = -\sum_{i=0}^{255} p(i) \log_2 p(i)$$

其中 $p(i)$ 为灰度级 $i$ 出现的概率（$p(i) = 0$ 的项不计入求和）。

程序运行结果：

- 图像尺寸：432 × 430
- 总像素数：185760
- **信息熵 H = 6.6886 bit/pixel**
- 理论无损压缩极限：平均每个像素至少需要 **6.6886 bit**

这意味着无论采用何种无损压缩算法，每个像素的平均编码长度不可能低于 6.6886 bit。

#### 3. 16位深度下的最大无损压缩比

若该图像原始为16位深度：

- 原始编码：16 bit/pixel
- 熵编码极限：6.6886 bit/pixel
- **理论最大无损压缩比**：

$$CR = \frac{16}{H} = \frac{16}{6.6886} = 2.3921 : 1$$

即理论上最多可以将原始数据压缩到原来的 $\frac{1}{2.39}$ 左右。
